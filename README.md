# Movie-Industry-Investment-DSS
โปรเจกต์นี้มีวัตถุประสงค์เพื่อพัฒนา ระบบสนับสนุนการตัดสินใจ (Decision Support System) สำหรับผู้บริหารสตูดิโอภาพยนตร์ เพื่อประเมินความคุ้มค่าและความเสี่ยงในการลงทุนผ่านการพยากรณ์รายได้ (Revenue Prediction) โดยใช้เทคนิค Machine Learning แบบการถดถอย (Regression) และการจำลองสถานการณ์ความเสี่ยง

# Tools 
**Language:** Python
**Libraries:** XGBoost, Scikit-learn, Pandas, NumPy, Streamlit (UI)
**Model:** XGBoost Regressor & Monte Carlo Simulation
**Techniques:** Target Encoding, Log Transformation, What-If Analysis

# Dataset
TMDB + IMDB Merged Movies Dataset from Kaggle :  
[https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud/data](https://www.kaggle.com/datasets/ggtejas/tmdb-imdb-merged-movies-dataset)
